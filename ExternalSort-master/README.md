# ExternalSort
A simple external sort </br>
Put integers in File "T1.txt" and seprate by space.</br> 
Use command line to put runsize and runsize range should be greater than 1 and less than free memory size</br>
The process of external sort is read the runsize of integers</br> 
Sort the input number of runsize and store it into File "T3.txt" and "T4.txt"</br>
merge two files until all number into one file
